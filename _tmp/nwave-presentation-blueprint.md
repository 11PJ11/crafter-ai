## Design Principles

### The Matrix as Argument

The Matrix isn't decoration; it's the critique itself:

| Matrix Element | What It Represents | The Truth |
|----------------|-------------------|-----------|
| Green falling code | AI marketing hype | "AI will replace you" |
| The illusion | Productivity promises | Feels fast, actually slower |
| The Matrix itself | The industry narrative | Wealth extraction disguised as progress |
| Being unplugged | nWave reality | Human-first, discipline-first |
| Red pill | The choice | See the system for what it is |

**Critical:** The audience must draw their own conclusions. We show evidence; they connect dots.

### Engagement Architecture

| Tool | Purpose | Signal |
|------|---------|--------|
| **Mentimeter** | Quick polls, temperature checks | QR code on slides |
| **Miro Board** | Parking lot for deep questions | "Park it here" |
| **ELMO** | "Enough, Let's Move On" | Respectful brake |

### Visual Theme

- **Primary palette:** Matrix green (#00FF41) on black
- **Accent:** Red pill red (#FF0000), Blue pill blue (#0000FF)
- **Typography:** Monospace for "code," clean sans-serif for content
- **Animation style:** Digital rain, glitch effects, terminal aesthetics
